# Розширена РГР

## Топологія мережі - 9 нод

```
        Node1   Node2   Node3
            \    |    /
             \   |   /
               HubA (центральна нода зірки 1)
                 |
              Bridge (міст між зірками)
                 |
               HubB (центральна нода зірки 2)
             /   |   \
            /    |    \
        Node4   Node5   Node6
```

## Запуск

1. **Запустіть CA сервер** (перший термінал):

   ```bash
   node ca-server.js
   ```

2. **Запустіть автоматичну демонстрацію** (другий термінал):
   ```bash
   node network-setup.js
   ```

## Сценарії демонстрації

1. **Пряме повідомлення**: Node1 -> HubA (в межах однієї зірки)
2. **Маршрутизація в зірці**: Node1 -> Node3 (через HubA)
3. **Маршрутизація між зірками**: Node1 -> Node6 (через HubA -> Bridge -> HubB)
4. **Broadcast**: Node2 розсилає повідомлення всім нодам
5. **Фрагментація**: Велике повідомлення Node4 -> Node1
6. **Повідомлення від Bridge**: Broadcast від з'єднувальної ноди
